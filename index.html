Git 整理

1. git 远程强制覆盖本地 / git 本地强制覆盖远程

  1. git 远程覆盖本地

    远程分支名记得替换下，一般都是在dev 或者版本号上开发。

    git fetch 只是下载远程的库的内容，不做任何的合并 git reset 把 HEAD 指向刚刚下载的最新的版本

  1. git 本地强制覆盖远程

    git 本地分支强制覆盖远程 master 分支。远程分支名记得替换下。
