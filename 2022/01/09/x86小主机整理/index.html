x86小主机整理

联想1L小主机简单汇总

联想M710q

  • 淘宝/咸鱼价格：
  • USB-A * 2（正面），DP * 2（背面）
  • 官方 BIOS 支持第六、七代处理器，可以刷 BIOS 上第八、九代处理器，以及上如 qnct 这一类的测试版CPU（教程可上 chh）
  • 主板是 b250 芯片组
  • 铜制散热器

联想M720q

  • 淘宝/咸鱼价格：￥800左右

  • USB-A + USB-C(3.1非全功能)（正面），DP * +HDMI（背面）

  • 主板是 b360 芯片组

  • 官方 BIOS 支持第六、七代处理器，可以刷 BIOS 上第八、九代处理器

  • 带 PCI 扩展槽

  • 单 M.2 插槽（仅配置一个）

    加装第二 M.2 槽：因为B360pcie通道受限，只能支持到sata协议的固态硬盘，无法支持nvme固态硬盘。

联想M910q

  • 淘宝/咸鱼价格：￥左右
  • USB-A + USB-C(3.1非全功能)（正面），DP * +HDMI（背面）
  • 主板是 b360 芯片组

M910q比M710q多了一个M.2的接口，其他都是一样的

联想M920q

  • 淘宝/咸鱼价格：￥xx
  • USB-A + USB-C(3.1非全功能)（正面），DP * +HDMI（背面）
  • 主板是 q370 芯片组
  • 官方 BIOS 支持第六、七代处理器，可以刷 BIOS 上第八、九代处理器，以及上如 qnct 这一类的测试版CPU（教程可上 chh），
  • 铜制散热器
  • 带 PCI 扩展槽，不可扩展显卡
  • 单 M.2 插槽（仅配置一个）
  • 相比 M720q 支持雷电3扩展，扩展成本在1000左右

联想M920x

  • 淘宝/咸鱼价格：￥1200-1400

  • USB-A + USB-C(3.1非全功能)（正面），DP * +HDMI（背面）

  • 主板是 q370 芯片组

  • 官方 BIOS 支持第六、七代处理器，可以刷 BIOS 上第八、九代处理器

  • 带 PCI 扩展槽，可扩展显卡或4口网卡

  • 相比 M720q 支持雷电3扩展，扩展成本在1000左右

  • 双 M.2 插槽

联想M72

  • 2/3代 CPU
  • H61主板

联想M73

  • 4代 CPU
  • H81主板
